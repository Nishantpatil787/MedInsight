# MedInsight AI

MedInsight is an AI-powered web application that helps doctors and patients quickly understand medical reports.

## Features

* Upload medical report PDF
* AI extracts diseases, medicines, allergies
* Detect abnormal lab values
* Clean dashboard visualization

## Tech Stack

Frontend:

* HTML
* CSS
* JavaScript

Backend:

* Python
* Flask

AI:

* OpenAI API
